# Mistral API Configurator

This is a simple web page that allows users to input their API key, chat endpoint, and completion endpoint. The data is stored in the browser's session storage.

## How to Use

1. Open the `index.html` file in your web browser.
2. Enter your HTML key, chat endpoint, and completion endpoint in the respective fields.
3. Click the "Save" button to store the data in your browser's session storage.

## Files

- `index.html`: The main HTML file.
- `styles.css`: The CSS file for styling.
- `script.js`: The JavaScript file for handling form submission and session storage.
